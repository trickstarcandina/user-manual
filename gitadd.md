"git add ."
Câu lệnh trên sẽ add
+ Các thay đổi trên các file đã tồn tại sẵn.
+ Các file mới được tạo ra.

"git add -u ."
Câu lệnh trên sẽ add
+ Các thay đổi trên các file đã tồn tại sẵn.
+ Các file đã bị xóa.

"git add -A"
Câu lệnh trên sẽ add cả 3 loại:
+Các thay đổi trên các file đã tồn tại sẵn.
+Các file mới được tạo ra.
+Các file đã bị xóa.
